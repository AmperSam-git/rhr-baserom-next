-- Graphics
Copies of unmodified graphics files.

-- Levels
Contains copies of the latest levels exported from your ROM and any time-stamped
copies if you ran a backup.

-- Lists
Contains modified list files for various tools that are imported when
the baserom is initialized.

-- Map16
Contains copies of the latest AllMap16 exported from your ROM and any time-stamped
versions if you ran a backup.

-- Palettes
Contains copies of the latest Shared palette exported from your ROM and any time-stamped
versions if you ran a backup.

-- ROM
Will contain timestamped ROM backups if you use the backup script.
