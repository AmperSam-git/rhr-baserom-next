### ExGFX61: Global ExAnimation
Contains optional frames for One-Ways (overrides the cloud graphics) and for the Infinite throw block. See the Demo level ExAnimation settings for more details

### ExGFXE17: Custom Global Block Tiles
All the tiles for the custom global blocks found in the Demo Level. This ExGFX resource is a drop in replacement for GFX17 that overrides the Big Bush and Large Slanted Pipe tiles.

### ExGFXFF: Font
This is a basic 8x8 font and symbol set, used in the Demo levels for in-game text.
