-------------
Baserom Files
-------------

RHR4.bps - the baserom itself is contained in this patch
RHR4.m16 - internal regular sprite tiles for Lunar Magic
RHR4.s16 - internal custom sprite tiles for Lunar Magic
RHR4.dsc - custom block descriptions as seen in Lunar Magic (generated by GPS)

@backup_baserom.bat - build system script for backing up your hack
@build_baserom.bat - build system script for running the tools to build your hack
@restore_from_backup.bat - build system script for restoring bits from backup
- see the "How to use the Build Scripts.txt" document for more details

.gitignore - as the baserom is managed on github, this file is a list of files to tell git to ignore
