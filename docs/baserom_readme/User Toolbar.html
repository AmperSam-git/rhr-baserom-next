User Toolbar

Lunar Magic has a built-in function to allow users to extend its toolbar with additional buttons (see the Lunar Magic help documentation if you need more details) and the baserom makes use of this to add some simple buttons that are some quick shortcuts to things.

ButtonNameDescription
Open Current ROM DirectoryOpens the current project directory in File Explorer
Open PIXI DirectoryOpens PIXI's directory in File Explorer
Edit Sprite ListOpens PIXI's sprite list in default text editor
Launch CFG EditorLaunches the sprite config editor tool that comes with PIXI
Open GPS DirectoryOpens GPS's directory in File Explorer
Edit Custom Block ListOpens GPS's block list in default text editor
Open UberASMTool DirectoryOpens UberASMTool's directory in File Explorer
Edit UberASM listOpens the UberASM list in default text editor
Open AddMusicK DirectoryOpens AddMusicK's directory in File Explorer
Edit Music ListOpens AddMusicK's custom music list in default text editor
