Baserom Directores

Build Tool Folder

This "buildtool" folder is the "Callisto", which is the tool used by the baserom to build it. Here you can configure Callisto itself, as well as any build options for the baserom.

Documentation Folder

All the build resources for you project are in here. Patches, initial setup files, map16, palettes, etc.

Resources Folder

All the build resources for you project are in here. Patches, initial setup files, map16, palettes, etc.

Shared Folder

This folder contains files that are shared between tools and resources, typically variable definitions and configuration files.

Tools Folder

The tools folder is where all the baserom tools are kept as well as any files and resources they use. If you are adding things to your project, you are adding them to the folders in this folder.

Workspace Folder

The workspace folder is where your project lives and is the output directory for the build process. It will contain your rom, its ExGraphics, and any supporting files.
