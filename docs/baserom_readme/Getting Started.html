Getting Started

Provide a ROM

First things first, you'll need to provide a clean copy of a headered, USA version Super Mario World ROM renamed to 'clean.smc' (take care to ensure the file extension* is .smc and not .sfc). Place this file in the "workspace" folder.

*this may require you to turn on extensions in Windows' File Explorer.

Initialize the Baserom

To start using the baserom you will first have to initialize the project. You can do this by running the provided PowerShell script: !initialize_baserom.ps1. When you run this script it will prompt you with some options:

1. Download and Setup all Baserom Tools
2. Restore Baserom list files.
3. Setup Baserom's custom Lunar Magic toolbar.
0. Exit

The only required step is: Download and Setup all Baserom Tools. Running this will check for all of the tools required by the baserom, and download them on demand. This is done to keep the baserom pretty lean and avoid distributing a lot of executables and binary files.

Building the Baserom

To make life easier for you as a hacker, this baserom comes with a build tool called Callisto that will run all the tools needed to build and update your hack everytime you want to make changes to it It will also monitor your project to keep things up-to-date. You can find it by running callisto.exe in the "callisto" folder.

See the Callisto readmes in the docs folder for more information.
