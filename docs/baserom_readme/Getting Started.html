Getting Started

Provide a ROM

First things first, you'll need to provide a clean copy of a headered, USA version Super Mario World ROM renamed to 'clean.smc' (take care to ensure the file extension* is .smc and not .sfc). Place this file in the "resources" folder.

*this may require you to turn on extensions in Windows' File Explorer.

Initialize the Baserom

To start using the baserom you will first have to initialize the project folder with all the tools. You can do this by running the provided PowerShell script: !initialize_baserom.ps1. This can be done in Windowe by either right clicking on it and choosing "Run" or executing it from a Terminal or the Command Line.

When you run this script it will prompt you with some options:

1. Download and Setup all Baserom Tools
2. Restore Baserom list files.
3. Setup Baserom's custom Lunar Magic toolbar.
0. Exit

Option 1 will check for all of the tools required by the baserom, and download them on demand. This is done to keep the baserom pretty lean and avoid distributing a lot of executables and binary files.

Option 2 will restore the pre-configured list files for the tools in the baserom, if you don't do this step the blocks and sprites and things will fail to be inserted.

Option 3 is optional, but it will install the a custom user toolbar that will display some useful buttons in Lunar Magic.

Building the Baserom

This baserom comes with a build tool called "Callisto" that will perform all actions needed to build and update your hack everytime you want to make changes. It executes all the common tools behind the scenes with a single command. Callisto will also monitor your project and export changes to keep things up-to-dat (provided you are using it's Edit functionality).

You can start it by running callisto.exe in the "callisto" folder. After which you will see a short menu, to build the baserom:

Rebuild (R)

If everything has been set-up correctly, Callisto will build the baserom in its entirety to be ready for use.

See the Callisto readmes in the docs folder for more information.
